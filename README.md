# Random Youtube Video (ryv)

## installation
```bash
virtualenv env
pip3 install -r requirements.txt
```

## execution
```bash
./app.py
```

## inspiration

[simple flask tutorial](https://flask.palletsprojects.com/en/2.0.x/quickstart/)
